# 💳 QR Payment Generator (Python)

A simple Python project that generates UPI payment QR codes for multiple payment platforms. Users can enter their own UPI ID and instantly generate QR codes for easy digital payments.

## ✨ Features

- 📱 Generate UPI QR Codes
- 💰 Supports:
  - PhonePe
  - Paytm
  - Google Pay
  - Navi
- ⚡ Fast and easy to use
- 🖼️ Automatically displays the generated QR code
- 🐍 Built with Python

## 🛠️ Technologies Used

- Python
- qrcode
- Pillow (PIL)

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/vishalrana45/qr_generator_py.git
```

Install the required libraries:

```bash
pip install qrcode pillow
```

## ▶️ How to Run

```bash
python takepaymentonline.py
```

Enter your UPI ID when prompted, choose the payment app, and the program will generate and display your QR code.
